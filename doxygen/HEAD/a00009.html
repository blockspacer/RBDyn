Detailed Description

Computation of the Coriolis effects matrix on a multibody.

The Coriolis factorization is not unique, we use the formulation of M. Bjerkend and K. Pettersen in "A new Coriolis matrix factorization", 2012 NB: ForwardDynamics::C() directly computes the product of this matrix with qd. This C*qd is unique, but C itself is not.
